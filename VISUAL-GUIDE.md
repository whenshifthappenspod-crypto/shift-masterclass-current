### **Each Episode Card Contains:**

**1. Thumbnail Image (16:9 aspect ratio)**
- Guest photo or interview screenshot
- View count badge in bottom-right corner
- Example: "👁️ 234K views"

**2. Episode Info**
- **Title:** Bold, white text (e.g., "Raoul Pal's Crypto Predictions")
- **Guest:** Smaller, gray text (e.g., "Raoul Pal - Macro Expert")

**3. Hover Effect**
- Card lifts up (translateY -8px)
- Border changes from gray to orange
- Shadow intensifies
- Thumbnail image scales up slightly (1.05x)
- Smooth 0.3s transition

**4. Click Action**
- Opens YouTube channel in new tab
- Cursor shows pointer (clickable)

## ✨ **Animation Details**

**On Page Load:**
- Cards fade in as you scroll
- Uses Intersection Observer
- Smooth fade-in-up animation

**On Hover (Episode Cards):**
- Transform: translateY(-8px) in 0.3s
- Border color: gray → orange
- Shadow: subtle → prominent
- Image scale: 1.0 → 1.05

**On Hover (YouTube Preview):**
- Transform: scale(1.02) in 0.3s
- Border: gray → orange with glow
- Overlay: opacity 0 → 1
- Play button: scale on hover

**On Click:**
- No animation (instant navigation)
- Opens link in new tab
